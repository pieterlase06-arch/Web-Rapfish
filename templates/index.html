RAPFISH MDS

Multi-Dimensional Scaling for Fisheries Sustainability Analysis

Cara Penggunaan

1. Unggah File: Pilih file Excel (.xlsx/.xls) atau CSV yang berisi data skor perikanan Anda.

2. Konfigurasi Baris: Tentukan di baris mana data dimulai dan berakhir (biasanya melewati baris header).

3. Konfigurasi Kolom:

  • Fisheries Names: Kolom yang berisi nama perikanan/sampel.
  • Attributes: Kolom yang berisi skor penilaian atribut.

4. Preview Data: Gunakan tombol Preview Data untuk memastikan rentang yang Anda pilih sudah benar.

5. Jalankan Analisis: Klik Run Rapfish Analysis untuk memproses MDS dan simulasi Monte Carlo.

6. Unduh Hasil: Klik Export Excel untuk mendapatkan laporan lengkap dalam format spreadsheet.
